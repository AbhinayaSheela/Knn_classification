# Knn_classification

This project demonstrates K-Nearest Neighbors (KNN) Classification using the Iris dataset from Scikit-learn.

## Algorithm Used
- K-Nearest Neighbors (KNN)

## Dataset
- Iris Dataset

## Steps Performed
- Imported required libraries
- Loaded dataset
- Applied preprocessing using StandardScaler
- Split dataset into training and testing sets
- Created KNN classification model
- Trained the model
- Predicted output values
- Evaluated model performance

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## Hyperparameter Tuning
Applied GridSearchCV using the following hyperparameters:
- n_neighbors
- weights
- algorithm
- leaf_size
- p

## Distance Measures Used
- Euclidean Distance
- Manhattan Distance
- Minkowski Distance

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Conclusion
This project demonstrates how KNN classification works by finding the nearest neighbors and predicting the majority class. Hyperparameter tuning improves model performance and accuracy.
